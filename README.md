# test-dataplus

Prueba técnica DathaPlus. Programación orientada a objetos

Este proyecto está desarrollado en **Java** y puede ejecutarse localmente siguiendo los pasos a continuación.

## Requisitos previos

Asegúrate de tener instalado lo siguiente:

- **Java JDK 17** o superior
- **Git**
- (Opcional) **Maven** o **Gradle** si el proyecto usa uno de estos gestores de dependencias.

## Clonar el repositorio

Abre una terminal y ejecuta:

```bash
git clone https://github.com/danielsamx/test-dataplus.git
```
